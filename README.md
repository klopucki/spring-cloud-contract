# spring-cloud-contract
Project created for LDI conference.

Project shows how to use Spring Cloud Contract.


## Installation

### First Producer
```
mvn install
```
Now you can see generated tests -> target-> generated-source-test

### Second Consumer
```
mvn test
```
You can see that all tests passed.
